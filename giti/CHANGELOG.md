# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-26

### Added
- Initial release
- License generation for 11 popular open-source licenses
- Interactive CLI mode for license selection
- Dependency license compatibility checker
- License statistics and distribution analysis
- Support for MIT, Apache-2.0, GPL-3.0, GPL-2.0, BSD-3-Clause, BSD-2-Clause, ISC, AGPL-3.0, LGPL-3.0, MPL-2.0, and Unlicense
- Automatic year and author insertion in license templates
- Color-coded output for better readability
- Compatibility matrix for license checking
- Warning system for potential license issues

### Features
- `generate` command - Create license files
- `list` command - Show all available licenses
- `info` command - Get details about specific licenses
- `check` command - Verify dependency compatibility
- `stats` command - Analyze license distribution
